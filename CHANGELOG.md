# Changelog

All notable changes to this project will be documented in this file.

## [1.0.0] - 2025-12-08

### Added
- **Strict 3-Color Palette**: Implemented a global design system enforcing Orange (#FB923C), Black (#0B0B0F), and Metallic Gray (#4B5563).
- **Typewriter Effect**: Updated B2C Hero typewriter text to use the brand orange accent for consistency with B2B pages.

### Changed
- **Color System**: Replaced all legacy colors (Green, Teal, Blue, Sky, etc.) with the new 3-color palette across all components (`Unternehmen.tsx`, `Ressourcen.tsx`, `ItHealthDashboard.tsx`, `servicesData.ts`).
- **Visual Assets**: Updated illustrations (`NIS2DocumentsIllustration`, `B2CHero`) to match the new color scheme.

### Removed
- **Fake Content**: Removed placeholder "Experten-Team" section from `Unternehmen.tsx`.
- **Fake Testimonials**: Removed placeholder "Kundenstimmen" section from `Privatkunden.tsx`.
